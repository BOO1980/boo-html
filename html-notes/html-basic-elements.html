Basic HTML Elements

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document
  • The <html> element is the root element of an HTML page
  • The <head> element contains meta information about the HTML page
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph
  • An HTML element is defined by a start tag, some content, and an end tag
  • Some HTML elements have no content (like the <br> element). These elements are called empty elements. Empty elements do not have an end tag!
  • HTML5 introduce more descriptive HTML tags. These include:
    • main
    • header
    • footer
    • nav
    • article
    • video
    • section
  • HTML elements have opening tags like <h1> and closing tags like </h1>
  • The text for an element goes between its opening and closing tag
  • These tags give a describe structure to your HTML, making your HTML easier to read
  • The tags also help with SEO (Search Engine Optimization) and accessibility
  • The main HTML5 tag helps search engines and other developers find the main content of your page

  • Commenting allows you to leave messages without affecting the browser display
  • Commenting also allows you to make code inactive
  • A comment in HTML starts with <!-- and end with -->
  • Elements inside other elements are call nesting
  • Nested elements should be indented to make it easier for your code to read
  • You can add images to your website by using the <img> element.
